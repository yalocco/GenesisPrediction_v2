# 0. Purpose

このドキュメントは  
LABOS の **事業モデルの方向性** を記録する。

目的

- 何を価値として提供するかを明確にする
- 収益化を「無理な商売」ではなく自然な形で設計する
- GenesisPrediction と LABOS の関係を整理する
- 将来の公開方針をAIと人間が共有できるようにする

---

# 1. Positioning

LABOS は  
**個人AI研究所の公開面** である。

GenesisPrediction が内部の研究・観測・分析システムであり、  
LABOS はその成果を外部へ見せる窓口となる。

関係

```text
GenesisPrediction
↓
研究 / 観測 / 分析
↓
LABOS
↓
公開 / 共有 / 小規模収益化
````

---

# 2. Core Value

LABOS が提供する価値は

```text
危険を早く知るための情報
```

である。

これは単なるニュース配信ではない。

目的は

* 家族を守る
* 資産を守る
* 世界の変化を理解する
* 危機の前兆を早く知る

ことである。

つまり LABOS は

**Risk Intelligence Service**

として位置づける。

---

# 3. Primary Audience

最初の対象は大衆ではない。

優先順位

1. 自分
2. 家族
3. 信頼できる友人
4. 少人数コミュニティ
5. 必要なら一般公開

LABOS は
最初から大規模サービスを目指さない。

---

```

公開面

候補

目的

目的

研究継続費の確保
```

LABOS は

```text
小さな研究所
```

である。

単なる利益最大化ではない。

* 家族を支える
* 将来に残せる仕組みを作る
* 小さくても自立した研究所を育てる

つまり LABOS は

```text
研究を続けるための事業
```

である。
