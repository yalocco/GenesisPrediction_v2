# 1. Project Overview

GenesisPrediction は

```

世界情勢観測
↓
構造分析
↓
未来シナリオ生成
↓
リスク予測

```

を目的とする **研究システム**である。

このプロジェクトは

```

Human + AI

```

による共同研究として運用される。

---

# 2. System Layers

GenesisPrediction は次のレイヤー構造を持つ。

```

Observation Layer
↓
Trend Layer
↓
Signal Layer
↓
Scenario Layer
↓
Prediction Layer

```

重要な原則

```

Prediction は主役ではない

```

主役は

```

Trend
Signal
Scenario

```

である。

Prediction は **最終要約**として生成される。
