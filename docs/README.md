# Docs Guide (GenesisPrediction v2)

このディレクトリは **GenesisPrediction v2 の「人間のための記憶・運用台本」** です。  
コードや自動処理とは役割を分け、  
**「どう使うか」「どう考えるか」「何を信頼するか」** を固定します。

---

## 📘 各ドキュメントの役割

### runbook.md（最重要）

* 毎日の正式運用手順（凍結）
* GUI / CLI を **どの順で回すか**
* やらないこと（事故防止ルール）

👉 迷ったら **まず runbook.md** を読む

---

### observation.md

* 日付ごとの観測ログ
* 自動生成 + 人間の1行所感
* 「当たり外れ」ではなく **見え方の変化** を残す

👉 思考の履歴・振り返り用

---

### gui_f1_panel_spec.md

* GUI 1画面パネル（F-1）の設計書
* 実装前の合意・凍結用

👉 GUIを触る前に読む

---

### assumptions.md

* 前提条件・世界観
* 分析が立脚している暗黙ルール

---

### diff_schema.md

* diff / daily_summary の構造説明
* データを見るときの仕様書

---

### fragile_points.md

* 壊れやすい点・注意点
* 触らない方がいい場所

---

## 🧭 運用原則（要約）

* `.md` は **人間のため**に書く
* 自動生成はヒントまで
* 判断・所感は人間が確定
* 迷ったら「観測 → 仮説 → 判断」に戻る

---

## 📝 更新ルール

* 小さく書く（1行でいい）
* 毎日書く必要はない
* 書かない日があっても問題なし
